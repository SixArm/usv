# Styles

USV styles that can customize output so it looks the way you want:

* Braces: use braces, such as "{US}" for Unit Separator.

* Controls: use zero-width controls, such as "\u{001F}" for Unit Separator.

* Symbols: use full-width symbols, such as "␟" for Unit Separator.

* Liners: use liners wrapping every symbol, such as "\n␟\n" for Unit Separator.

* Sheets: use layouts like spreadsheet sheets, such as "␟" for Unit Separator and "␟\n" for Record Separator.
